# mini-quant-demo

這是我在自學量化交易時練習寫的一個小程式。  
我想了解「投資決策如何被數據化」，所以用 Python 做了一個最簡單的交易策略：  
比較 5 日與 20 日移動平均線的交叉，觀察價格趨勢的變化。

---

## 💡 內容說明
- 使用 Python 與 pandas 處理歷史價格資料  
- 以雙均線（MA5 / MA20）作為進出場條件  
- 簡單計算策略報酬率，用來了解量化交易的基本流程
